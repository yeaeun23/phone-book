# phone-book

- 개발 100%
- 디자인 100%
- DB설계 80%

## 1. 기능

- 연락처 보기
- 연락처 정렬하기 (등록/이름순)
- 연락처 검색하기 (분류, 이름, 전화번호, 이메일, 비고)
- 연락처 추가하기 (분류, 이름, 전화번호, 이메일, 비고)
- 연락처 수정/삭제하기
